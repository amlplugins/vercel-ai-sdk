# @amlplugins/vercel-ai-sdk

Native `ai` SDK re-exported under the `@amlplugins` namespace.

> Vercel AI SDK — provider-agnostic LLM toolkit with generateText, streamText, generateObject, tool use, agents.

## Install

```bash
npm install @amlplugins/vercel-ai-sdk
```

## Use

```ts
import { /* named exports from ai */ } from "@amlplugins/vercel-ai-sdk";
// or
import sdk from "@amlplugins/vercel-ai-sdk";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
